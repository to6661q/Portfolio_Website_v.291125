# 👋 Welcome to My Project Portfolio Website

Hi there! I'm **Toriq As Syarif**, and welcome to the repository for my this project. 

I created this website as my digital space to showcase my work, share a bit about my journey as a developer, and connect with others in the tech community. It's a place where I document the projects I'm proud of and the skills I'm constantly sharpening.

## 🌟 What You'll Find Here
This isn't just a static resume; it's a reflection of my design and development sensibilities. Here are some of the key features I've implemented:

*   **Clean & Modern UI:** I wanted to create a minimalist, user-friendly interface that looks great and feels smooth to use on any device.
*   **Custom Sidebar Navigation:** I built a dedicated sidebar component to make navigating through different sections (like Projects, Skills, and Contact) seamless and intuitive.
*   **Project Showcase:** A dedicated section to highlight my best work, complete with descriptions, the tech stacks I used, and links to live demos.
*   **Fully Responsive Design:** I made sure every element adapts perfectly, whether you're viewing it on a wide desktop monitor or a mobile phone.
*   **Type-Safe Codebase:** I wrote the entire frontend using TypeScript to ensure the code is robust, maintainable, and less prone to silly runtime errors.

## 🛠️ Tech Stack
I chose modern, industry-standard tools to build this portfolio, ensuring it's fast, scalable, and a joy to work with:

*   **Framework:** React / Next.js 
*   **Language:** TypeScript (100%)
*   **Styling:** Tailwind CSS
*   **UI Components:** shadcn/ui (for accessible and beautifully designed components like the sidebar)
*   **Deployment:** Vercel *(Adjust if you use Netlify or something else)*
